# Simulación - Bebedero ON/OFF

Simulación interactiva de un sistema de control ON/OFF para un bebedero automático. La aplicación permite visualizar en tiempo real el nivel de agua, el error de control, y simular el comportamiento del sistema con diferentes parámetros configurables.

## Instalación y Ejecución

### Paso 1: Clonar el repositorio

Clona el repositorio en tu máquina local usando el siguiente comando:

```bash
git clone https://github.com/juavazquez/tdc-simulacion.git
```

### Paso 2: Navegar al directorio

Abre el explorador de archivos de tu sistema operativo y navega hasta el directorio donde clonaste el repositorio.

### Paso 3: Ejecutar la aplicación

1. Localiza el archivo `index.html` en el directorio del proyecto
2. Haz doble clic en el archivo `index.html`
3. El archivo se abrirá automáticamente en tu navegador predeterminado

¡Listo! La aplicación debería estar funcionando y podrás interactuar con la simulación.

## Uso

- **Perro bebe (sorbo)**: Simula un sorbo individual que reduce el nivel de agua
- **Perro bebe (de a poco)**: Simula múltiples sorbos en rápida sucesión
- **Reset**: Reinicia la simulación al estado inicial
- **Controles deslizantes**: Ajusta los parámetros del sistema en tiempo real
